# Class, Object, Method and Variable

## Class

A **Class** is a blueprint (template) used to create objects.

It defines the properties (data) and behaviors (methods) of an object.

### Example

A **Car** is a class.

The class contains common properties such as:

* Brand
* Color
* Speed
* Brake
* Accelerator

---

## Object

An **Object** is an instance of a class.

Each object has its own values for the properties defined in the class.

### Example

From the **Car** class, we can create multiple objects such as:

* BMW
* Audi
* Tesla

All these objects belong to the same class but contain different values.

---

## Method (Function)

A **Method** is a block of code that performs a specific task.

Methods are used to execute operations.

### Example

* `addNumbers()`
* `displayDetails()`
* `startEngine()`

---

## Variable

A **Variable** is used to store data.

Variables hold values that can be used throughout the program.

### Example

```java
int speed = 100;
String color = "Red";
```

---

## Relationship

```text
Class
   │
   ▼
Object
   │
   ▼
Methods & Variables
```

---

## Key Points

* A Class is a blueprint or template.
* An Object is an instance of a class.
* Methods perform specific tasks.
* Variables are used to store data.
* One class can have multiple objects.
