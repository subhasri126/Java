# History of Java

## Introduction

Java was developed by **James Gosling** and his team at **Sun Microsystems**.

It was officially released in **1995** with the goal of creating a platform-independent programming language.

Later, **Oracle Corporation** acquired Sun Microsystems and now maintains Java.

---

## Timeline

* **Developer:** James Gosling
* **Organization:** Sun Microsystems
* **First Release:** 1995
* **Current Maintainer:** Oracle Corporation

---

## Key Points

* James Gosling is known as the creator of Java.
* Java was developed at Sun Microsystems.
* Java was first released in 1995.
* Oracle Corporation currently maintains Java.
