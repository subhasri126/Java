#  Learning Goals

* Understand the fundamentals of Java.
* Learn how Java programs are compiled and executed.
* Understand the roles of JDK, JRE, and JVM.
* Learn the Java execution flow from source code to machine code.
* Understand Class Loaders and the Delegation Hierarchy.
* Learn how the JIT Compiler improves program performance.
* Build a strong foundation before moving to advanced Java topics.
