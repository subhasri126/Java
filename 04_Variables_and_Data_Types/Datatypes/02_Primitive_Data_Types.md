# Primitive Data Types

## Introduction

Primitive Data Types are the built-in data types provided by Java. They are used to store simple values directly in memory. Primitive data types are predefined by the Java language and are the foundation for storing data in Java programs.

Java provides **8 Primitive Data Types**.

---

## Characteristics of Primitive Data Types

- They are built into Java.
- They store values directly in memory.
- They are faster than reference data types.
- They have a fixed size.
- They are used to store simple values such as numbers, characters, and boolean values.

---

## Types of Primitive Data Types

Java has the following 8 primitive data types:

| Data Type | Category | Size |
|-----------|----------|------|
| byte | Integer | 1 Byte |
| short | Integer | 2 Bytes |
| int | Integer | 4 Bytes |
| long | Integer | 8 Bytes |
| float | Floating Point | 4 Bytes |
| double | Floating Point | 8 Bytes |
| char | Character | 2 Bytes |
| boolean | Logical | JVM-dependent |

---

## Classification of Primitive Data Types

Primitive data types can be classified into four categories:

### 1. Integer Data Types
Used to store whole numbers.

- byte
- short
- int
- long

### 2. Floating Point Data Types
Used to store decimal numbers.

- float
- double

### 3. Character Data Type
Used to store a single Unicode character.

- char

### 4. Boolean Data Type
Used to store logical values.

- boolean

---

## Summary

- Java provides **8 Primitive Data Types**.
- Primitive data types directly store values.
- They have fixed memory sizes.
- They are classified into Integer, Floating Point, Character, and Boolean data types.
